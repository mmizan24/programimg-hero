    CSS styling -External

    Key points of CSS styling -External.

    • CSS stands for Cascading Style Sheets.
    • CSS is used to style and layout web pages.
    • CSS can control the color, font, spacing, and positioning of elements on a webpage.
    • External CSS is written in a separate file with a .css extension and linked to HTML documents using the tag.
    • Using external CSS allows for easier maintenance and consistency across multiple web pages.
    • CSS rules are made up of selectors and declarations, where selectors target HTML elements and declarations define the styles to be applied.

    Here adding some more features of externa css styles
    • CSS supports various units of measurement, including pixels (px), ems (em), rems (rem), percentages (%), and viewport units (vw, vh).
    • CSS allows for responsive design through media queries, enabling web pages to adapt to different screen sizes and devices.
    • CSS frameworks like Bootstrap and Tailwind CSS provide pre-designed styles and components to speed up web development.
    • CSS variables (custom properties) allow for easier management of colors, fonts, and other design elements across a stylesheet.
